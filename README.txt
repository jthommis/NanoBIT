The following scripts are intended for the analysis of NanoBiT data.
	NanoBiT_analysis_individual_replicates.R	Analysis of n=1 experiments
	NanoBiT_analysis_multiple_replicates.R		Analysis of 2 or 3 biological replicates (can easily be adjusted for the analysis of more biological replicates)
See the notes (comments) in the scripts for more information and guidelines on how to use them!

	annotation_TEMPLATE.xlsx			template that can be used to create the plate annotation file
	annotation_TEMPLATE.xlsm			same thing but with a handy export to csv button
